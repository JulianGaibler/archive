## Introduction

A Platform to upload videos, images and GIFs so that they can be archived and organized to be found more easily.

## Client

``` bash
# Install dependencies
npm install
# Run dev-server
npm run dev
# Build client
npm run build
```

## Server

``` bash
# Install dependencies
npm install
# Run dev-server
npm run dev
# Build server
npm run build
```

## Database

``` bash
# in main directory
docker-compose up -d
# in server directory
npm run knex migrate:latest
```
